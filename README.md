# Calculadora de IMC utilizando JavaScript | HTML & CSS
## Objetivo do projeto!
Criação de uma página web responsiva capaz de calcula o IMC (Indice de massa corporea) com base em 2 informações,
Peso e Altura (Kg e Metros).
